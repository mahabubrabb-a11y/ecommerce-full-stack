# E-Commerce Full Stack Application

## Technologies
- React.js
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Features
- User Authentication
- Product Management
- Shopping Cart
- Order Management
- Payment Integration
- Responsive Design

## Installation

```bash
npm install
cd client
npm install
```

## Run Backend

```bash
npm start
```

## Run Frontend

```bash
cd client
npm start
